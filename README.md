## ⚙️ Core Technical Architecture
* **Backend Runtime:** Node.js, Express REST API routing.
* **Database Management:** Cloud PostgreSQL hosted on Railway.
* **Security Layer:** JWT (JSON Web Tokens) security headers with Bcrypt password hashing algorithms.
* **User Interface:** Simple vanilla semantic HTML, CSS styling, and synchronized JavaScript execution engines.

## 🛠️ Operational Setup Guide
1. Run `npm install` to synchronize core app software engines.
2. Setup variable values inside a local environment `.env` file.
3. Run `node initDb.js` once to generate standard system tables dynamically on PostgreSQL.
4. Spin up the monitoring workspace environment locally by typing `npm run dev`.
